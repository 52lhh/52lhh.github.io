计算日期间时间差

  1. 当项目开发时，会遇到需要根据起始时间和结束时间计算时间差，如果直接利用两时间相减，发现是行不通的

  2. 而通常情况下，时间格式也不尽相同，如：2018-05-19T08:04:52.000+0000、yyyy-mm-dd hh:mm:ss、yyyy/mm/dd hh:mm:ss，因此直接相减是无法达到目的

  3. 因此需要通过计算两时间的毫秒数之差后，将得到的毫秒数进行转化即可

    const beginTime = '2020-03-23 18:25:56'
const endTime = '2020-03-28 09:42:34'

const time1 = new Date(beginTime.replace(/-/g, '/')) // replace将-转化为/
const time2 = new Date(endTime.replace(/-/g, '/'))

// 时间差的毫秒数
const diffDate = time1.getTime() - time2.getTime()
