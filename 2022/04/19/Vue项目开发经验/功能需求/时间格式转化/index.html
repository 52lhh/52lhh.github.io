时间格式转化

  1. 在项目开发中，会遇到请求某接口后，接口返回的时间格式为“2018-05-19T08:04:52.000+0000”，这种格式通常不会直接渲染到页面上，因此就需要将该时间格式转化为“yyyy-mm-dd hh:mm:ss ”

  2. 代码处理如下

    const beforeDate = "2018-05-19T08:04:52.000+0000"

const d = new Date(beforeDate) // 将beforeDate转化为日期对象d
const date = d.getFullYear() + '-' + 
      (d.getMonth() + 1 < 10 ? "0" + (d.getMonth() + 1) : d.getMonth() + 1) + '-' +
      (d.getDate() < 10 ? "0" + d.getDate() : d.getDate()) + ' ' +
      (d.getHours() < 10 ? '0' + d.getHours() : d.getHours()) + ':' + 
      (d.getMinutes() < 10 ? '0' + d.getMinutes() : d.getMinutes()) + ':' + 
      (d.getSeconds() < 10 ? '0' + d.getSeconds() : d.getSeconds())

// getFullYear(): 获取日期对象d的年份
// d.getMonth(): 获取日期对象d的月份-1
// d.getDate(): 获取日期对象d的日期-1
// d.getHours(): 获取日期对象d的小时数
// d.getMinutes(): 获取日期对象d的分钟数
// d.getSeconds(): 获取日期对象d的秒数

// 将以上时间组成部分进行拼接，并判断月份、日期、小时数、分钟数、秒数值是否大于10，如小于10，则拼接0，避免转化后的时间格式为“2018-5-19 8:0:0"”
