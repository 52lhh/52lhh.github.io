HTML简介

HTML编写要求

  1. 什么是html

    1. HyperText Markup Language：超文本标记语言，网页语言
      • 超文本：超出文本的范畴，使用html可以轻松实现这样的操作
      • 标记：html所有的操作都是通过标记实现的，标记就是一个标签<标签语言>
      • 网页语言
    2. 一个HTML程序
      • 创建Java文件.java
        • 先编译，后运行(JVM虚拟机)
      • HTML后缀是.html或.htm
        • 直接通过浏览器就可以运行
      • 代码
        • 这是我的<font size="5" color="red">第一个html程序!</font>

  2. html的规范(遵循)

    1. 一个html文件开始标签和结束标签:
    2. html包含两部分内容
      • <head> 设置相关信息 </head>
      • <boby> 显示在页面上的内容都写在boby里面 </boby>
    3. html的标签有开始标签，也要有结束标签
      • 例：<head></head>
    4. html的代码不区分大小写
    5. 有些标签，没有结束标签，在标签内结束
      • 比如：换行 </br> </hr>

        文档类型声明标签

  3. 文档类型声明标签

    1. <!DOCTYPE>：文档类型声明，作用就是告诉浏览器使用哪种HTML版本来显示网页。如：<!DOCTYPE html>
      • 这句话的代码意思是：当前也页面采取的是HTML5版本来显示网页
    2. 注意：
      • <!DOCTYPE>声明位于文档中的最前面的位置，处于<html>标签之前
      • <!DOCTYPE>不是一个HTML标签，它就是文档类型声明标签

  4. lang语言种类

    1. en定义语言为英语
    2. Zh-CN定义语言为中文
    3. 简单来说定义为en就是英文网页；
    4. 定义为zh-CM就是中文网页，其实对于文档显示来说，定义成en的文档也可以显示z中文，定义成zh-CN的文档也可以显示英文
    5. 这个属性对浏览器和搜索引擎还是有作用的

  5. 字符集

    1. <meta charset="UTF-8"/>
    2. UTF-8被称为万国码

  6. HTML操作思想（重点）

    1. 网页中有很多数据，不同的数据可能需要不同的显示效果，这个时候就需要使用标签把操作的数据包起来（封装起来），通过修改标签的属性值实现标签内数据样式的变化
    2. 一个标签相当于一个容器，想要修改容器内数据的样式，只需要改变容器的属性值，就可以实现容器内数据样式的变化

      HTML中常用的标签

      段落和换行标签

  7. 段落标签：可以把HTML文档分割为若干段落
    <p> 文本内容 </p>

  8. 换行标签：强制换行
    <br/>

    文本格式化标签

  9. 标签语义：突出重要性，比普通文字更重要

    语义 标签
    字体加粗 <strong></strong>
    字体倾斜 <em></em>
    字体删除线 <del></del>
    字体下划线 <ins></ins>

  10. <div>和<span>标签

    1. div：表示分割，分区，占一行（相当于一个大包）
    2. span：跨度，跨距（相当于一个小包）

      文字标签和注释标签

  11. 文字标签：修改文字的样式

    1. <font></font>
    2. 属性:
      • size：文字的大写 取值范围：1-7；若超出7，还是默认7
      • color：文字颜色
        • 两种表示方式
        • 英文单词：red green blue black white yellow gray..
        • 使用十六进制数表示，通过工具实现不同的颜色

  12. 注释标签

    1. java注释有三种
    2. html的注释：<!--html注释内容-->

      标题标签、水平线标签和特殊字符

  13. 标题标签
    <h1></h1> <h2></h2> <h3></h3> .. <h6></h6>

    1. 从h1到h6，标题依次变小，同时会自动换行

  14. 水平线标签
    <hr/>

    1. 属性
      • size：水平线的粗细，取值范围1-7
      • color：颜色
    2. 代码
      <hr size="5" color="blue"/>

  15. 特殊字符

    1. 想要在页面上显示这样的内容<html>:是网页的开始！
    2. 需要对特殊字符进行转义
      特殊字符 转义
      < &lt;
      > &gt;
      空格 &nbsp;

      列表标签

  16. 自定义标签（重点）<dl> </dl>：表示列表的范围

    1. 在dl里面 <dt> </dt>：上层内容
    2. 在dl里面 <dd> </dd>：下层内容
    3. <dl></dl>里面只能包含<dt>和<dd>
    4. <dt>和<dd>个数没有限制，经常是一个<dt>对应多个<dd>

  17. 有序列表 <ol> </ol>：有序列表的范围

    1. 属性 type：设置排序方式（1（默认），a，i）
    2. 在ol标签里面<li>具体内容</li>

  18. 无序列表(重点) <ul> </ul>：无序列表的范围

    1. 无序列表的各个列表项之间没有顺序级别之分，是并列的
    2. <ul></ul>中只能嵌套<li></li>，直接在<ul></ul>标签中输入其他标签或者文字的做法是不被允许的
    3. <li>与</li>之间相当于一个容器，可以容纳所有元素
    4. 无序列表会带有自己的样式属性，但在实际使用时，我们会使用CSS来设置
    5. 属性 type：实心圆disc(默认) 空心圆circle 实心方块square
    6. 在ul里面<li>具体内容</li>
    7. 去掉无序列表的项目符号：list-style: none;

      图像标签（重点）

  19. 图像标签
    <img src="图片路径" width="宽度" height="高度"/>

    src 类型 图片的路径
    alt 文本 替换文本，图像无法显示时用文字替换
    title 文本 提示文本，鼠标放到图像之上后 ，显示的文件
    border 像素 设置图像的边框粗细
    width 像素 图片的宽度
    height 像素 图片的高度

    路径的介绍（重点）

  20. 路径分为：绝对路径和相对路径

    1. 绝对路径：盘符地址或网页地址
    2. 相对路径：一个文件相对于另外一个文件的位置

      超链接标签（重点）

  21. 链接资源

    1. <a href="链接到的资源路径" target="打开方式"> 显示在页面中的内容</a>
      • href：链接的资源的地址
      • target：设置打开的方式，默认是在当前页打开
        • _blank：在一个新窗口打开
        • _self：在当前页打开
          `<a href="HelloWorld.html" target="_blank">这是一个超链接</a>`
    2. 当超链接标签href中不加任何内容，则打开文件所在文件夹路径
    3. 当超链接不需要到任何的地址，在href里加#

  22. 链接分类

    1. 外部链接
    2. 内部链接：网站页面之间的相互连接，直接链接内部页面名称即可
    3. 空链接：如果当时没有确定链接目标时，首页
    4. 下载链接：如果href里面地址是一个文件或者压缩包，就会下载这个文件
    5. 网页元素链接：网页中的 各种网页元素，如文本，表格，音频，视频等都可以添加超链接
    6. 锚点链接（重点）：点击锚点链接时，可以快速定位到页面中某个位置
      • 在链接文本的href属性中，设置属性值为#名字的形式，如<a href="#two">第2集</a>
      • 找到目标位置标签，在其中添加id属性 = 名字，如：<h3 id="two">第2集介绍</h3>

  23. 定位资源

    1. 定位资源：定义一个位置
      <a name="top"> 顶部 </a>
    2. 回到这个位置
      <a name="#top"> 顶部 </a>
    3. 引入一个标签
      <pre> </pre>：原样输出

      表格标签（重点）

  24. 可以对数据进行格式化，使数据显示的更清晰

  25. 表格的写法

    1. 首先定义一个表格的范围使用<table></table>
      • 定义一行使用一个<tr></tr>
      • 定义一个单元格使用一个<td></td>
    2. 操作技巧

  26. <table></table>：表示表格的范围

     1. 例：`<table border="1" bordercolor="black" cellspacing="0" width="500" height="300">`

  27. 表格标签属性

    1. align：表格在页面中的位置，属性值(left center right)
    2. border：表格线
    3. bordercolor：表格线的颜色
    4. cellpadding:单元格与其内容之间的距离
    5. cellspacing：单元格之间的距离
    6. width：表格的宽度
    7. height：表格的高度

  28. 在table里面<tr></tr>

    1. 设置对齐方式：align(left center right)
    2. 在tr里面<td></td>
             设置显示方式：align(left center right)
    3. 使用<th></th>也可以表示单元格：表示实现居中和加粗

  29. 表格结构标签

    1. <thead></thead>：用于定义表格的头部，存放表头
    2. <tbody></tbody>：用于定义表格的主题，存放主体

  30. 表格的标题

       `<caption></caption>`

  31. 合并单元格

    1. 跨行：rowspan
      • 将需要合并的那一行的其他不需要的信息全部删掉
        <tr><th rowspan="3">人员信息</th></tr>
            删除多余单元格
    2. 跨列：colspan
      • 将需要合并的那一 列的其他不需要的信息全部删掉
              `<th colspan="3">人员信息</th>`
      • 删除多余单元格

        表单标签（重点）

  32. <form></form>：定义一个表单的范围

    1. 属性：
      • action：提交到地址，默认提交到当前的页面
      • method：表单的提交方式
        • 常用的有两种 get和post，默认是get请求
      • enctype：一般请求下不需要，做文件上传时需要设置这个属性
    2. 面试题：get和post的区别
      • get请求地址栏会携带提交的数据，post不会携带（在请求体里面，http协议时讲）
      • get请求安全级别较低，post较高
      • get请求数据大小有限制，post没有限制
    3. 输入项：可以输入内容或者选择内容的部分
    4. 大部分的输入项使用<input type="输入项的类型"/>
    5. 在输入项里面必需要有一个name属性(name属性：当前input表单的名字，后台可以通过name属性找到这个表单，name的主要作用就是用来区别于不同的表单)

  33. input输入表单元素

    1. 普通输入项：<input type="text"/>
      • maxlength：规定输入字段中的最大长度
    2. 密码输入项：<input type="password"/>
    3. 单选输入项：<input type="radio"/>
      • 在里面需要属性name
      • 若需要两个单选输入项互斥，将两者的name属性值设置相同即可
      • name的属性值必须要相同
      • 必须有一个value值
      • 实现要默认选中的属性 
              `checked="checked"`
    4. 复选输入项：<input type="checkbox"/>
      • 在里面需要属性name
      • name的属性值必须要相同
      • 必须有一个value值
      • 实现默认选中的属性 
              `checked="checked"`
    5. 文件输入项(在后面上传时候用到)：<input type="file"/>
    6. 隐藏项（不会显示在页面上，但是存在于html代码里面） 
             `<input type="hidden"/>`
    7. 提交按钮 
             `<input type="submit"/>`
       `<input type="submit" value="注册"/>`
    8. 使用图片提交 
             `<input type="image" src="图片路径"/>`
    9. 重置按钮：回到输入项的初始状态 
             `<input type="reset"/>`
    10. 普通按钮（和js一起使用） 
            `<input type="button" value=""/>`

  34. select下拉表单元素

    1. 下拉输入项(不是在input标签里面的)： 
        <select name="birth">  
    <option value="1991">1991</option>  
    <option value="1992">1992</option>  
    <option value="1993">1993</option>  
  </select>
      • 实现默认选中的属性
        selected="selected"

  35. 方法：select() — 让文本框内的文字处于选定状态

    1. 语法：input.select();

  36. textarea文本域元素

    1. 文本域 
      `<textarea clos="20" rows="10">..</textarea>`

  37. 案例：使用表单标签实现注册页面案例：见IDEA

    2. 超链接实现页面效果

    4. 如果表格里面的单元格没有内容，使用空格作为占位符$nbsp;

      label标签

  38. <label>：标签为input元素定义标注（标签）

  39. <label>的作用：增加用户体验，当点击<label>标签中的文字等任意内容，就会选中该表单控件所关联的按钮或表单

    1. 注意：需要让表单中的id值等于label标签中的for值

  40. 语法：
    <label for = "sex">男</label>
    <input type = "radio" name = "sex" id = "sex"/>

    • 核心：<label>标签的for属性应当与相关元素的id属性相同
