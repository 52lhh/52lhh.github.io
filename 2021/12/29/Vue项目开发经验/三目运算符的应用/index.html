三目运算符的应用

  1. 在vue开发中经常会遇到当遇到不同情况需要取不同值的情况，但使用if语句又过于麻烦，此时就可以使用三目运算符

  2. 情景一：在html中使用三目运算符

    1. 当需要根据某个变量值的改变而决定html中文本的内容时

       <p>{{ item.relationship === '01' ? '父母' : item.relationship === '02' ? '子女' : item.relationship === '03' ? '夫妻' : '其他亲属'}}</p>

  3. 情景二：在开发时，由于项目测试经常会切换正式环境和测试环境，因此相关环境的参数就需要来回切换。此时，来回的复制粘贴不仅很麻烦，还很容易弄错，因此可以使用三目运算符进行优化

    /**
 * 运行环境，release为正式环境，test为测试环境
 */
// 此时只需要改变EVN的值为test或release即可
const EVN = 'test'

let common = {
    URL_MOHRSS_URL: EVN === 'test' ? 'https://test-ssc.mohrss.gov.cn' : 'https://ssc.mohrss.gov.cn', // 地址
    ECARD_AK: EVN === 'test' ? '8c907cf7a36a47bf8a2002144205caf5' : '5b5310e9b0d30828db78f6f24685c3e8', // 8c907cf7a36a47bf8a2002144205caf5 测试 5b5310e9b0d30828db78f6f24685c3e8 正式
    ECARD_APPID: EVN === 'test' ? '4f17da03a2944cbdb3a4d11aee7fc9e5' : '64ba3fed939d42a9a10aa647fceefec5', // appid 4f17da03a2944cbdb3a4d11aee7fc9e5 测试 64ba3fed939d42a9a10aa647fceefec5 正式
    EPIDEMIC: EVN === 'test' ? 'http://smt-stg.yun.city.pingan.com/dongguan/stg/app/feature/electronic-card/#/?memberCode=' : 'https://igj.dg.cn/app/feature/electronic-card/#/?memberCode=', // 路径
}
