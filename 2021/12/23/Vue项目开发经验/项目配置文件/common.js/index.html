common.js通用配置文件

  1. 封装common.js的作用：在开发中将需要使用到的公共函数和公共参数进行封装，以后在其他组件中需要使用时直接调用即可

  2. 封装方法

    1. 创建common.js文件，在文件中引入Vue

      import Vue from 'vue'

    2. 将公共函数和公共参数进行封装

      let common = {
    // 公共参数
    API_IV_KEY: '***',
      API_SK_KEY: '***', 
      MERCHANT_NO: ***, 
    
    // 公共函数
    outF1 () {
        console.log("这是一个公共的函数")
    },
    outF2 () {
        console.log("这是一个公共的函数")
    }
}

    3. 将common.js文件导出

      export default common

  3. 使用common.js文件：在main.js入口文件中写以下代码

    // 引入api.js文件
import api from './***/common'

Vue.prototype.$common = common

  4. 之后在开发中需要使用这些封装好的公共函数和参数时，直接使用this.$common.参数名或者**this.$common.函数名()**即可
