精灵图、字体图标、用户界面

精灵图的使用

  1. 精灵图主要针对于小的背景图片使用
  2. 主要借助于背景位置来实现
    • background-position
  3. 一般情况下精灵图都是负值（网页中的坐标：X轴右边走是正值，左边走是负值；Y轴同理）

字体图标

字体图标的特点

  1. 字体图标的优点
    1. 轻量级：一个图标字体要比一系列的图像要小，一旦字体加载，图标就马上会渲染出来，减少服务器的请求
    2. 灵活性：本质为文字，可以随意更改颜色，产生阴影，透明效果，旋转等
    3. 兼容性：几乎是支持所有的浏览器
  2. 注意：字体图标不能代替精灵技术，只是对工作中图标部分技术的提升和优化
  3. 总结：如果遇到一些结构和样式比较简单的小图标就使用字体图标

字体图标的下载

  1. icomoon字库
  2. 阿里iconfont字库

字体图标的使用与引入

  1. 把下载包里面的fonts文件夹放入页面根目录下
  2. 在CSS样式中全局声明字体：简单理解把这些字体文件通过css引入到我们页面中；要注意字体文件路径的问题
    • 文件路径问题：路径需要根据字体图标文件夹icomoon中的style.css文件调用font文件夹中四个文件的路径而改变
      @font-face {
    font-family: 'icomoon';
    src: url('fonts/icomoon.eot?8axf2s');
    src: url('fonts/icomoon.eot?8axf2s#iefix') format('embedded-opentype'),
        url('fonts/icomoon.ttf?8axf2s') format('truetype'),
        url('fonts/icomoon.woff?8axf2s') format('woff'),
        url('fonts/icomoon.svg?8axf2s#icomoon') format('svg');
    font-weight: normal;
    font-style: normal;
    font-display: block;
}
  3. html标签内添加字体图标
    <!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    @font-face {
      font-family: 'icomoon';
      src: url('fonts/icomoon.eot?8axf2s');
      src: url('fonts/icomoon.eot?8axf2s#iefix') format('embedded-opentype'),
        url('fonts/icomoon.ttf?8axf2s') format('truetype'),
        url('fonts/icomoon.woff?8axf2s') format('woff'),
        url('fonts/icomoon.svg?8axf2s#icomoon') format('svg');
      font-weight: normal;
      font-style: normal;
      font-display: block;
    }

    .one {
      /* font-family与上方导入的代码中的第一行需要保持相同 */
      font-family: 'icomoon';
      font-size: 50px;
      color: pink;
    }

    .two {
      /* font-family与上方导入的代码中的第一行需要保持相同 */
      font-family: 'icomoon';
      font-size: 50px;
      color: skyblue;
    }
  </style>
</head>

<body>
  <span class="one"></span>
  <span class="two"></span>
</body>
</html>

字体图标追加

  1. 将压缩包内的selection.json重新上传即可，然后选中自己想要的新的图标，重新下载压缩包，并替换原来的文件
  2. 步骤
    1. 打开icomoon字库网址
    2. 点击左上角import icons按钮，弹出对话框，找到selection.json文件，打开，点击重新加载YES
    3. 开始选择新的字体图标，选择完毕后点击右下角generate font按钮，点击右下角download按钮下载

通过类名调用字体图标（推荐） (页首)

  1. HTML页面中引入字体图标中css文件 — style.css
    • 注意修改style.css文件中的四个路径，修改方法见上
  2. 标签直接调用图标对象的类名即可（类名在style.css文件中标注，或直接打开dome.html文件查询）

CSS的小三角

  1. 利用CSS语法创建小三角
    1. 语法：
        .tri {
      height: 0;
        width: 0;
        line-height: 0;
        font-size: 0;
        border: 10px solid transparent;
        border-left-color: pink;
    }
      • transparent：透明的意思

用户界面

鼠标样式 cursor

  1. 设置或检索在对象上移动的鼠标指针采用何种系统预定义的光标形状
  2. 语法：
    • cursor: 属性值
  3. 属性值
    属性值 描述
    default 小白 默认
    pointer 小手
    move 移动
    text 文本
    not-allowed 禁止

表单轮廓线

  1. 给表单添加outline: 0; 或者outline:none; 样式之后，就可以去掉默认的蓝色边框
  2. 语法
    • input {outline: none}

表单轮廓线

  1. 给表单添加outline: 0; 或者outline:none; 样式之后，就可以去掉默认的蓝色边框
  2. 语法
    • input {outline: none}

防止拖动文本域

  1. 实际开发中，文本域右下角是不可以拖动改变大小的
  2. 语法
    • textarea {resize: none;}

vertical-align 行内块元素和文字垂直居中

  1. CSS的vertical-align属性使用场景：经常用于设置图片或者表单（行内块元素）和文字垂直对齐
  2. 官方解释：用于设置一个元素的垂直对齐方式，但是他只针对于行内元素或者行内块元素有效
  3. 语法
    • vertical-align: baseline | top | middle | bottom
      值 描述
      baseline 默认，元素放置在父元素的基线上
      top 把元素的顶端与行中最高元素的顶端对齐
      middle 把此元素放置在父元素的中部
      bottom 把元素的顶端与行中最低的元素顶端对齐

解决图片底部空白缝隙问题

  1. bug：图片底侧会有一个空白缝隙，原因是行内块元素会和文字的基线对齐
  2. 解决方法
    1. 给图片添加
      • vertical-align: top | middle | bottom；（提倡使用）
    2. 把图片转换为块级元素display: block;

溢出文本用省略号显示

单行文字溢出省略号显示

  1. 步骤分为三步
    1. 如果文字显示不开也要强制一行内显示
      • white-space: nowrap;（这里的默认属性值为normal，即换行显示）
    2. 将文字溢出部分隐藏起来
      • overflow: hidden;
    3. 文字溢出的时候用省略号来显示
      • text-overflow: ellipsis;
  2. 代码
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .box {
            width: 150px;
            height: 30px;
            background-color: skyblue;
            /* 1.如果文字显示不开也要强制一行内显示 */
            white-space: nowrap;
            /* 2.将文字溢出部分隐藏起来 */
            overflow: hidden;
            /* 3.文字溢出的时候用省略号来显示 */
            text-overflow: ellipsis;
        }
    </style>
</head>
<body>
    <div class="box">
        朝辞白帝彩云间，千里江陵一日还
    </div>
</body>
</html>

布局技巧

margin负值应用

  1. 相邻的两个带边框的盒子，若两个盒子紧挨，会导致盒子紧挨着的边框相接，出现1+1=2的问题
  2. 解决方法，将其中一个盒子的margin值改为负值，使相接的边框重叠，避免问题发生
  3. 鼠标经过某个盒子时，边框显示亮起
  4. 方法一：在大盒子没有加相对定位的情况下，则可以给小盒子加相对定位
  5. 方法二：鼠标经过某个盒子时，提高当前盒子的层级z-index: 1;
    <!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
<style>
   * {
       margin: 0;
       padding: 0;
   }
   ul li {
       position: relative;
       float: left;
       list-style: none;
       width: 200px;
       height: 300px;
       border: 1px solid red;
       margin-left: -1px;
   }
   ul li:hover {
       /* 方法一: 在大盒子没有加相对定位的情况下，则可以给小盒子加相对定位 */
       /* position: relative;
       border: 2px solid black; */
       /* 方法二: 鼠标经过某个盒子时，提高当前盒子的层级 */
       z-index: 1;
       border: 2px solid black;
   }
</style>
</head>
<body>
<ul>
   <li></li>
   <li></li>
   <li></li>
   <li></li>
</ul>
</body>
</html>

文字围绕浮动元素

  1. 文字会围绕图片显示，因为创造浮动的初衷就是做文字环绕效果的
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        * {
            margin: 0;
            padding: 0;
        }
        .box {
            width: 270px;
            height: 70px;
            background-color: pink;
            margin: 10px auto;
            padding: 5px;
        }
        .pic {
            float: left;
            width: 120px;
            height: 70px;
            margin-right: 5px;
        }
        img {
            width: 100%;
            height: 100%;
        }
    </style>
</head>
<body>
    <div class="box">
        <div class="pic">
            <img src="../images/bear.jpg" alt="">
        </div>
        <p>熊大熊大，不好啦，光头强又来砍树啦！</p>
    </div>
</body>
</html>

CSS三角妙用

  1. 制作斜边三角形
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .box {
            width: 0;
            height: 0;
            border-color: transparent red transparent transparent;
            border-style: solid;
            border-width: 30px 15px 0 0;
        }
    </style>
</head>
<body>
    <div class="box"></div>
</body>
</html>
