CSS的三大特性、盒子模型、圆角边框与阴影

发表于 2021-09-20 更新于 2021-09-21 分类于 CSS

学习笔记：CSS之三大特性、盒子模型、圆角边框与阴影
个人笔记可能会存在一些错误，您在阅读中如若发现，欢迎您的指正

CSS的三大特性

  1. CSS的三大特性：层叠性，继承性，优先级

层叠性（覆盖性）

  1. 相同的选择器给设置相同的样式，此时一个样式就会覆盖（层叠）另一个冲突的样式。层叠性主要解决样式冲突的问题
  2. 层叠性原则：
    1. 样式冲突，遵循的原则是就近原则，那个样式距离结构近，就执行哪个样式
    2. 样式不冲突，不会层叠

继承性

  1. CSS中的继承：子标签会继承父标签的某些样式，如文本颜色和字号
  2. 恰当的使用继承可以简化代码，降低CSS样式的复杂性
  3. 行高的继承性
    1. 代码
      body {
  font：12px/1.5 Microsoft YaHei;
}
    2. 行高可以跟单位也可以不跟单位
    3. 如果子元素没有设置行高，则会继承父元素的行高为1.5，此时子元素的行高是：当前子元素的文字大小*1.5
    4. body行高1.5这样写法最大的优势就是里面子元素可以根据自己的文字大小自动调整行高

优先级

  1. 优先级等级
    选择器 选择器权重
    继承或* 0,0,0,0
    元素选择器 0,0,0,1
    类选择器 伪类选择器 0,0,1,0
    ID选择器 0,1,0,0
    行内样式 style=”” 1,0,0,0
    !important 重要的 无穷大

权重

  1. 权重有4组数字组成，可以叠加但是不会有进位。
  2. 可以理解为类选择器永远大于元素选择器，id选择器永远大于类选择器，以此类推..
  3. 等级判断从左到右，如果某一位数值相同，则判断下一位数值
  4. 继承而来的权重为0，如果该元素没有直接选中，无论父元素权重多高，子元素权重都为0

权重的叠加

  1. 权重叠加：如果是复合选择器，则会有权重叠加，需要计算权重
  2. 计算
    复合选择器 权重
    div ul li 0,0,0,3
    .nav ul li 0,0,1,2
    a:hover 0,0,1,1
    .nav a 0,0,1,1

盒子模型

盒子模型概述

  1. 用于封装周围的HTML元素，它包括：边框，外边距，内边距，和实际内容

外边框

  1. 在进行布局前需要把数据封装到一块一块的区域内，这个区域的专业术语叫做盒子
  2. 边框会影响盒子的实际大小，实际开发中盒子需要减去边框的宽度

边框的粗细

  1. 语法：border-width

边框的样式

  1. 语法：border-style
  2. 样式
    1. solid 实线边框
    2. dashed 虚线边框
    3. dotted 点线边框
    4. double 双线框
      • 两条边线框宽度的和加起来等于定义的border-width值

边框的颜色

  1. 语法：border-color

边框的复合写法

  • border:2px solid blue;
  1. 各边框分别设置
    1. 上：border-top
    2. 下：border-bottom
    3. 左：border-left
    4. 右：border-right

表格细线边框

  1. border属性控制浏览器绘制表格的方式，他控制相邻单元格的边框。
    1. 语法：border-collapse: collapse;
    2. 表示相邻边框合并在一起

内边距

  1. 如果盒子本身没有指定width/height属性，则此时padding不会撑开盒子大小

内边距padding

  1. 内边距会影响盒子的实际大小，实际开发中盒子需要减去内边距的宽度
  2. padding：上下左右四边距统一设置
    • padding: 20px;
  3. 各边距分别设置
    1. 上：padding-top
    2. 下：padding-bottom
    3. 左：padding-left
    4. 右：padding-right

padding的复合属性

  1. padding属性（简写属性）可以右一到四个值
    值的个数 表达意思
    padding:5px 1个值，代表上下左右都有5px像素内边距
    padding:5px 10px 2个值，代表上下内边距是5像素左右内边距是10像素
    padding:5px 10px 20px 代表上内边距5像素左右内边距10像素下内边距20像素
    padding:5px 10px 20px 30px 上是5像素右10像素下20像素左是30像素 顺时针

外边距

  1. margin属性用于设置外边距，即控制盒子和盒子之间的距离

外边距margin

  1. margin：上下左右四边距统一设置
    • margin: 20px;
  2. 各边距分别设置
    1. 上：margin-top
    2. 下：margin-bottom
    3. 左：margin-left
    4. 右：margin-right

margin复合属性

  1. margin属性（简写属性）可以右一到四个值
  2. 具体和padding相同

外边距典型应用——块级盒子水平居中

  1. 外边距可以让块级盒子水平居中，必须满足两个条件：
    1. 盒子必须指定宽度（width）
    2. 盒子的左右外边距都设置auto
  2. 语法：margin: 0 auto;
  3. 注意以上方法是让块级元素水平居中，行内元素或者行内块元素水平居中给其父元素添加text-align: center即可

嵌套块元素垂直外边距的塌陷

  1. 对于两个嵌套关系（父子关系）的块元素，父元素有上边距同时子元素也有上外边距，此时父元素会塌陷较大那个的外边距值
  2. 解决方案
    1. 为父元素定义上边框（会导致改变盒子大小）
    2. 为父元素定义上内边距（会导致改变盒子大小）
    3. 为父元素添加overflow: hidden（推荐）

清除内外边距

  1. 网页元素很多都有默认的内外边距，而且不同浏览器默认的也不一致，因此我们在布局前，首先要清除下网页元素的内外边距
  2. 语法：
     * {
   padding: 0;
   margin: 0;
 }
    • 以后编写CSS样式时的第一句代码
  3. 注意：行内元素为了照顾兼容性，尽量只设置左右内外边距，不要设置上下内外边距

圆角边框和阴影

圆角边框（重点）

  1. 语法：border-radius: length;
    1. 参数值可以为数值也可以为百分比
  2. radius半径（圆的半径）原理：圆或椭圆与边框的交集形成圆角的效果
  3. 如果是正方形，想要设置为一个圆，把数值修改为高度或者宽度的一半即可，或者写为50%
  4. 如果是一个矩形，设置高度的一半就可以设置一个圆角矩形
  5. 该属性是一个简写属性
    1. 可以跟四个值：分别代表左上角，右上角，右下角，左下角
    2. 可以跟两个值：第一个值代表左上角，右下角；第二个值代表右上角，左下角
  6. 若分开写
    1. 左上角：border-top-left-radius
    2. 右上角：border-top-right-radius
    3. 右下角：border-bottom-right-radius
    4. 左下角：border-bottom-left-radius

盒子阴影

  1. 盒子阴影的属性添加：box-shadow
  2. 语法：box-shadow: h-shadow v-shadow blur spread color inset;
  3. 属性
    值 描述
    h-shadow 必需 水平阴影的位置，允许负值
    v-shadow 必需 垂直阴影的位置，允许负值
    blur 可选 模糊距离（阴影实虚）
    spread 可选 阴影的尺寸（阴影大小）
    color 可选 阴影的颜色，参阅CSS颜色值
    inset 可选 将外部阴影（outset）改为内部阴影
    1. 注意：
      • 默认的是外部阴影（outset），但是不可以写这个单词，否则导致整个阴影无效
      • 盒子阴影不占用空间，不会影响其他盒子排列

盒子半透明

  1. 让盒子半透明的属性添加：opacity: (0 ~ 1);
  2. 例：opacity: 0.3;

文字阴影

  1. 文字阴影的属性添加：text-shadow
  2. 语法：text-shadow: h-shadow v-shadow blur color;
  3. 属性
    值 描述
    h-shadow 必需 水平阴影的位置，允许负值
    v-shadow 必需 垂直阴影的位置，允许负值
    blur 可选 模糊距离（阴影实虚）
    color 可选 阴影颜色，参阅CSS颜色值
