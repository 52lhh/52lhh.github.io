变量概述

什么是变量

  1. 变量是用于存放数据的容器，通过变量名获取数据，甚至数据可以修改

变量的本质

  1. 本质：变量是程序在内存中申请的一块用来存放数据的空间

声明变量

变量声明

var age; //声明一个名称为age的变量
  1. var是一个JS关键字，用来声明变量( variable变量的意思)。使用该关键字声明变量后，计算机会自动为变量分配内存空间，不需要程序员管

赋值

age = 10; //给age这个变量赋值为10
  1. = 是用来把右边的值赋给左边的变量空间中 此处代表赋值的意思
  2. 变量值是程序员保存到变量空间里的值

变量的初始化

var age = 18;
var name = '芦崽崽';

更新变量

  1. 一个变量被重新复赋值后，它原有的值就会被覆盖，变量值将以最后一次赋的值为准
    var age = 10;
    age = 18; //最后结果为18，因为10被覆盖了

同时声明多个变量

  1. 同时声明多个变量，只需要写一个var，多个变量名之间使用英文逗号隔开
    <script>
  var name = '芦崽崽',
      age = 18,
      sex = '女';
</script>

声明变量的特殊情况

  1. 只声明不赋值

    1. 结果是：undefined（未定义的）

  2. 不声明不赋值直接使用

    1. 报错

  3. 不声明直接赋值

    1. 可以使用，但会变为全局变量，不提倡

      <script>
// 只声明不赋值
var name;
console.log(name); //输出空 或 undefined

// 不声明不赋值直接使用
console.log(age); //报错 age is not defined

// 不声明直接赋值 
qq = 10;
console.log(qq); //可以使用，但会变为全局变量，不提倡
</script>

变量命名规范

  1. 由字母(A-Za-z)、数字(0-9)、下划线(_)、美元符号($)组成，如: usrAge，num01，_name
  2. 严格区分大小写；例：var app; 和var App; 是两个变量
  3. 不能以数字开头；例：18age是错误的
  4. 不能是关键字、保留字；例：var、for、while
  5. 变量名必须有意义
  6. 遵守驼峰命名法；首字母小写，后面单词的首字母需要大写；例：myFirstName
  7. 推荐翻译网站:有道，爱词霸

数据类型

简单数据类型

简单数据类型 说明 默认值
Number 数字型，包含整数型和浮点型 0
Boolean 布尔值类型，true（加法运算中当1看）和false（加法运算中当0看） false
String 字符串类型 “”
Undefined var a; 声明了变量a但是没有给值，此时a = undefined undefined
Null var a = null; 声明了变量a为空值 null

数字型进制

  1. 最常见的进制有二进制，八进制，十进制，十六进制
  2. 八进制：在数字前面加0；0~7
    Var num = 010; //转换为十进制就是8
  3. 十六进制：在数字前面加0x；09 af
    Var num = 0xa; //转换为十进制就是10

数字型范围

  1. JavaScript中数值的最大和最小值
    alert(number.max_value);   //1.7976931348623157e + 308
alert(number.min_value);   //5e - 324
    1. 最大值：number.max_value
    2. 最小值：number.min_value
  2. 数字型三个特殊值
    alert(infinity); 或 alert(number.max_value * 2);   //infinity
alert(-infinity); 或 alert(-number.max_value * 2);  //-infinity
alert(NaN);        //NaN
    1. infinity: 代表无穷大，大于任何数值
    2. -infinity:代表无穷小，小于任何数值
    3. NaN，Not a number，代表一个非数值
  3. isNaN()：用来判断一个变量是否为非数字的类型，返回true或者false
    1. 若该变量为数字，则返回flase
    2. 若该变量不为数字，则返回true

字符串型

  1. 字符串型可以是引号中的任意文本，语法为双引号和单引号
  2. 字符串引号嵌套（外双内单 外单内双）
  3. 字符串转义字符
    转义符 解释说明
    \n 换行符，n为newline的意思
    \ 斜杠\
    ' ‘单引号
    " “双引号
    \t tab缩进
    \b 空格，b是blank的意思
  4. 检测字符串长度 length
  5. 字符串拼接 +
  6. 多个字符串之间可以使用 + 进行拼接，其拼接方式为：字符串 + 任何类型=拼接之后的新字符串
  7. 拼接前会把与字符串相加的任何类型转成字符串，再拼接成一个新的字符串
    <script>
 // 字符串的拼接 + 只要有字符串和其他类型相拼接，最终的结果都是字符串类型
 console.log('沙漠' + '骆驼'); //'沙漠骆驼'
 console.log('芦崽崽' + 18); //'芦崽崽18'
 console.log('芦崽崽' + true); //'芦崽崽true'
 console.log(12 + 12); //24
 console.log('12' + 12); //'1212'
</script>
  8. 变量不要写到字符串内部，要通过和字符串相连接的方式实现

Undefined和Null

  1. 一个声明后没有被赋值的变量会有一个默认值undefined

  2. 相连或者相加之后的结果

    <script>
  // 一个声明后没有被赋值的变量会有一个默认值undefined
  var str;
  console.log(str); //undefined
  var variable = undefined;
  console.log(variable + 'pink'); //undefinedpink
  console.log(variable + 1); //NaN与undefined 和数字相加，最后的结果是NaN

  //null 空值
  var space = null;
  console.log(space + 'pink'); //'nullpink'
  console.log(space + 1); //1
</script>

  3. typeof：检测获取变量的数据类型

    <script>
  var num = 1;
  console.log(typeof num); //number
  var str = '芦崽崽';
  console.log(typeof str); //string
  var flag = true;
  console.log(typeof flaf); //boolean
  var und;
  console.log(typeof und); //undefined
  var i = null;
  console.log(typeof i); //object

  //prompt取过来的值是字符型的
  var age = prompt('请输入您的年龄');
  console.log(age);
  console.log(typeof age); //string
</script>

数据类型转换

转换为字符串

方式 说明 案例
toString() 转换为字符串 var num = 1; alert(num.toString());
String()强制转换 转换为字符串 var num = 1; alert(String(num));
加号拼接字符串 和字符串拼接的结果都是字符串 var num = 1; alert(num + “我是字符串”);
  1. toString()和String()使用方式不一样
  2. 三种转换方式,更喜欢用第三种加号拼接字符串转换方式，这一种方式也称之为隐式转换

转换为数字型（重点）

方式 说明 案例
parseInt(string)函数 将string类型转成整数数值型(可以用于去除单位) parseInt(‘78’) 
// 将string类型转成整数数值型
var str = '10.32';
console.log(parseInt(str)); //10 取整
// 可以用于去除单位
console.log(parseInt('12px')); //12
方式 说明
parseFloat(string)函数 将string类型转成浮点数数值型(可以用于去除单位) parseFloat(‘78.21’) 
// 将string类型转成浮点数数值型
var str1 = '10.24';
console.log(parseFloat(str1)); //10.24
// 可以用于去除单位
console.log(parseFloat('120px')); //120
方式 说明 案例
Number强制转换函数 将string类型转换为数值型 Number(‘12’) 
// 将string类型转换为数值型
var str2 = '123';
console.log(Number(str2)); //123
方式 说明 案例
js隐式转换(- * /) 利用算术运算隐式转换为数值 ‘12’-0 
// 利用算术运算隐式转换为数值
console.log('12' - 0); //12
console.log('120' / 10); //12
console.log('12' * 1); //12

转换为布尔型

方式 说明 案例
boolean()函数 其他类型转成布尔型 Boolean(‘true’)
  1. 代表空、否定的值都会被转换为false，如’’、0、NaN、null、undefined
  2. 其余值都会被转换为true

运算符

  1. 运算符(operator)也被称为操作符,是用于实现赋值、比较和执行算数运算等功能的符号
  2. JavaScript中常用的运算符有：
    1. 算术运算符
    2. 递增和递减运算符
    3. 比较运算符
    4. 逻辑运算符
    5. 赋值运算符

算术运算符

  1. 概念：算术运算使用的符号，用于执行两个变量或值的算术运算
    运算符 描述 实例
    ‘+’ 加 10 + 20 = 30
    ‘-‘ 减 10 - 20 = -10
    ‘*’ 乘 10 * 20 = 200
    ‘/‘ 除 10 / 20 = 0.5
    ‘%’ 取余数(取模) 返回除法的余数
  2. 表达式：由数字、运算符、变量等以能求得数值的有意义排列方法所得的组合

递增和递减运算符

  1. 递增（++）和递减（–）
  2. 在JavaScript中,递增(++)和递减(–)既可以放在变量前面,也可以放在变量后面
    1. 放在变量前面时，称为前置递增(递减)运算符
      • ++num：先自增（自减），后返回值
    2. 放在变量后面时，称为后置递增(递减)运算符
      • num++：先返回值，后自增（自减）
  3. 递增和递减运算符必须和变量配合使用

比较运算符

  1. 概念：比较运算符（关系运算符）是两个数据进行比较时所使用的运算符，比较运算后，会返回一个布尔值（true/false）作为比较运算的结果
    运算符名称 说明 案例 结果
    ‘<’ 小于 1 < 2 true
    ‘>’ 大于 1 > 2 false
    ‘>=’ 大于等于 2 >= 2 true
    ‘<=’ 小于等于 3 <= 2 false
    ‘==’ 判等（会转型） 37 == 37 true
    ‘!=’ 不等于 37 != 37 false
    ‘=== !==’ 全等（要求值和数据类型都一致） 37 === ‘37’ false
  2. 小结
    符号 作用 用法
    = 赋值 把右边给左边
    == 判断 判断两边值是否相等（注意此时有隐式转换）
    === 全等 判断两边的值和数据类型是否完全相等

逻辑运算符

  1. 概念：逻辑运算符是用来进行布尔值运算的运算符,其返回值是布尔值；后面开发中经常用于多个条件的判断
    1. &&：”逻辑与”，简称”与” and（一假则假，全真为真）
    2. ||：”逻辑或”，简称”或” or（一真则真，全假为假 ）
    3. !（取反符）：”逻辑非”，简称”非” not

短路运算（逻辑中断重点）

  1. 短路运算原理：当有多个表达式（值）时，左边的表达式值可以确定结果时，就不再继续运算右边的表达式的值
  2. 逻辑与短路运算
    1. 语法：表达式1 && 表达式2
      • 如果第一个表达式的值为真，则返回表达式2
      • 如果第一个表达式的值为假，则返回表达式1
  3. 逻辑或短路运算
    1. 语法：表达式1 || 表达式2
      • 如果第一个表达式的值为真，则返回表达式1
      • 如果第一个表达式的值为假，则返回表达式2

赋值运算符

  1. 基本赋值运算符：=
  2. 扩展的赋值运算符：+=，-=，*=，/=，%=
  3. 先执行等号右边再赋值给左边

运算符优先级

优先级 运算符 顺序
1 小括号 ()
2 一元运算符 ++ – !
3 算术运算符 先* / %，后+ -
4 关系运算符 > >= < <=
5 相等运算符 == != === !==
6 逻辑运算符 先&&，后
7 赋值运算符 =
8 逗号运算符 ,
