学习笔记：JavaScript数组
个人笔记可能会存在一些错误，您在阅读中如若发现，欢迎您的指正

数组

数组的创建

  1. 数组（Array）：一组数据的集合
  2. 数组里面的数据要用逗号进行分隔
  3. 数组中可以存放任意类型的数据，例如字符串，数字，布尔值等
  4. JS中创建数组有两种方式

利用new创建数组

var arr = new Array();

利用数组字面量创建数组（常用）

  1. []：表示数组的字面量
    var arr = [];

获取数组元素

  1. 数组的索引
    1. 索引（下标）：用来访问数组元素的序号（数组下标从0开始）
    2. 数组可以通过索引来访问、设置、修改对应的数组元素，可以通过数组名[索引]的形式来获取数组中的元素

遍历数组

  1. 遍历数组：把数组的元素从头到尾访问一次

数组长度

  1. 使用“数组名.length”可以访问数组元素的数量（数组长度）
  2. arr.length动态监测数组元素的个数

数组中新增元素

修改length长度

  1. 通过修改length长度来实现数组扩容目的
  2. length属性是可读写的
  3. 语法
    arr.length = 数组长度;

修改索引号

  1. 语法
    var arr = [1, 2, 3];
arr[3] = 4;
  2. 通过追加索引号，并为该索引号对应的元素赋值的方式新增元素
  3. 若索引号为已经存在，赋值则替换原来该索引位置的数组元素
  4. 不能直接给数组名赋值，否则会覆盖掉以前的数据

冒泡排序

var arr = [2, 5, 11, 85, 3, 16, 24, 9, 7];
  for (var i = 0; i < arr.length; i++) {
    for (var j = 0; j < arr.length - 1 - i; j++) {
      if (arr[j] < arr[j + 1]) {
        var temp = arr[j];
        arr[j] = arr[j + 1];
        arr[j + 1] = temp;
      }
    }
}
