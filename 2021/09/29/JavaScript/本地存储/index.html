本地存储

本地存储

  1. 随着互联网的快速发展，基于网页的应用越来越普遍，同时也变的越来越复杂，为了满足各种各样的需求，会经常性在本地存储大量的数据，HTML5规范提出了相关解决方案

本地存储特性

  1. 数据存储在用户浏览器中
  2. 设置、读取方便、甚至页面刷新不失数据
  3. 容量较大，sessionStorage约5M、localStorage约20M
  4. 只能存储字符串，可以将对象JSON.stringify()编码后存储

sessionStorage和localStorage

sessionStorage概述

  1. 生命周期为关闭浏览器窗口
  2. 在同一个窗口（页面）下数据可以共享
  3. 以键值对的形式存储使用

sessionStorage方法（重点）

  1. 存储数据：sessionStorage.setItem(key, value);

  2. 获取数据：sessionStorage.getItem(key);

  3. 删除数据：sessionStorage.removeItem(key);

  4. 删除所有数据：sessionStorage.clear();

    <body>
    <input type="text">
    <button class="set">存储数据</button>
    <button class="get">获取数据</button>
    <button class="remove">删除数据</button>
    <button class="clear">删除所有数据</button>

    <script>
        console.log(localStorage.getItem('uname'));

        var ipt = document.querySelector('input')
        var set = document.querySelector('.set');
        var get = document.querySelector('.get');
        var remove = document.querySelector('.remove');
        var clear = document.querySelector('.clear');
        // 存储数据
        set.addEventListener('click', function () {
            var v_set = ipt.value;
            sessionStorage.setItem('uname', v_set);
            sessionStorage.setItem('pwd', v_set);
        })

        // 获取数据
        get.addEventListener('click', function () {
            var v_get = sessionStorage.getItem('uname');
            console.log(v_get);
        })

        // 删除数据
        remove.addEventListener('click', function () {
            sessionStorage.removeItem('uname');
        })

        // 删除所有数据
        clear.addEventListener('click', function () {
            sessionStorage.clear();
        })
    </script>
</body>

localStorage概述

  1. 生命周期永久生效，除非手动删除，否则关闭页面也会存在
  2. 可以多窗口（页面）共享（同一浏览器可以1共享）
  3. 以键值对的形式存储使用

localStorage方法（重点）

  1. 存储数据：localStorage.setItem(key, value);

  2. 获取数据：localStorage.getItem(key);

  3. 删除数据：localStorage.removeItem(key);
