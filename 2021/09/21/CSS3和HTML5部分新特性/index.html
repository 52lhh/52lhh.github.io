CSS3和HTML5部分新特性

学习笔记：HTML5和CSS3部分新特性
个人笔记可能会存在一些错误，您在阅读中如若发现，欢迎您的指正

HTML5新特性

CSS初始化

  1. 不同浏览器对有些标签的默认值是不同的，为了消除不同浏览器对HTML文本呈现的差异，照顾浏览器的兼容，我们需要对CSS初始化
  2. 简单理解：CSS初始化是指重设浏览器的样式（也称为CSS reset）
  3. 每个网页都必须首先进行CSS初始化

HTML5新特性

  1. HTML5的新特性主要针对以前的不足，增加了一些新的标签，新的表单和新的表单属性

HTML5新增的语义化标签

  1. <header>: 头部标签
  2. <nav>: 导航标签`
  3. <article>: 内容标签
  4. <section>: 定义文档某个区域
  5. <aside>: 侧边栏标签
  6. <footer>: 尾部标签

HTML5新增的多媒体标签

  1. 新增的多媒体标签主要包含两个
    1. 音频：<audio>
    2. 视频：<video>
  2. 使用这些标签可以很方便的在页面中嵌入音频和视频，而不再去使用flash和其他浏览器插件
  3. 视频<video>
    1. 语法：<video src="文件地址" controls="controls"></video>
  4. 视频常见属性
    属性 值 描述
    autoplay autoplay 视频就绪自动播放（谷歌浏览器需要添加muted来解决自动播放的问题）
    controls controls 向用户显示播放控件
    width pixels（像素） 设置播放器宽度
    height pixels（像素） 设置播放器高度
    loop loop 播放完是否继续播放该视频，循环播放
    preload auto（预先加载视频）none（不加载视频） 规定是否预先加载视频（如果有了autoplay则忽略该属性）
    src url 视频url地址
    poster imgurl 加载等待的画面图片
    muted muted 静音播放
  5. 音频常见属性
    属性 值 描述
    autoplay autoplay 如果出现该属性，则音频在就绪后立即播放（谷歌浏览器不支持）
    controls controls 如果出现该属性，则向用户显示控件，比如播放按钮
    loop loop 如果出现该属性，则当音频结束时重新开始播放
    src url 要播放音频的URL
  6. 多媒体标签总结
    1. 音频标签和视频标签使用方式基本一致
    2. 浏览器支持情况不同
    3. 谷歌浏览器把音频和视频自动播放禁止 了
    4. 可以给视频标签添加muted属性来静音播放视频，音频不可以（通过JavaScript解决）
    5. 视频标签是重点，经常设置自动播放，不使用controls控件，循环和设置大小属性

  1. 重点记忆：number tel search这三个
  2. 属性值
    属性值 说明
    type="email" 限制用户输入必须为email类型
    type="url" 限制用户输入必须为url类型
    type="date" 限制用户输入必须为日期类型
    type="time" 限制用户输入必须为时间类型
    type="month" 限制用户输入必须为月类型
    type="week" 限制用户输入必须为周类型

HTML5新增表单属性

  1. 属性值

    placeholder 提示文本 表单的提示信息，存在默认值将不显示

    autocomplete off/on 当用户在字段开始键入时，浏览器基于之前键入过的值，应该显示出在字段中填写的选项；默认已经打开，如：autocomplete=”on”，关闭autocomplete=”off”需要放在表单内，同时加上name属性，同时成功提交

CSS3的新特性

属性选择器

  1. 属性选择器可以根据元素特定的属性来选择元素。可以不借助于类或者id选择器
    选择符 简介
    E[att] 选择具有att属性的E元素
    E[att="val"]（重点掌握） 选择具有att属性且属性值等于val的E元素
    E[att^=”val”] 匹配具有att属性且值以val开头的E元素
    E[att$="val"] 匹配具有att属性且值以val结尾的E元素
    E[att*="val"] 匹配具有att属性且值中含有val的E元素
  2. 属性选择器，类选择器，伪类选择器，权重都为10
    • 例：div[class^=icon]权重为11；其中div为标签选择器，权重为1；class^=icon为属性选择器，权重为10

结构伪类选择器

  1. 结构伪类选择器主要根据文档结构来选择元素，常用于根据父级选择器里面的子元素
  2. nth-child
    选择符 简介
    E:first-child 匹配父元素中的第一个子元素E
    E:last-child 匹配父元素中最后一个子元素E
    E:nth-child(n) 匹配父元素中的第n个子元素E
    1. 选择某个父元素的一个或者多个特定的子元素
      • n可以是数字，关键字和公式
      • n如果可以是数字，就是选择第n个子元素，里面数字从1开始
      • n可以是关键字：even偶数，odd奇数
      • n可以是公式：常见的公式如下（如果n是公式，则从0开始计算，但是第0个元素或者超出了元素的个数会被忽略）
        公式 取值
        2n 偶数
        2n+1 奇数
        5n 5的倍数
        n+5 从第5个开始（包含第五个）到最后
        -n+5 前5个（包含第5个）
  3. nth-of-type
    选择符 简介
    E:first-of-type 指定类型E的第一个
    E:last-of-type 指定类型E的最后一个
    E:nth-of-type(n) 指定类型E的第n个,公式同上
  4. 区别（重点）
    1. nth-child：对父类元素里面所有孩子排序选择（序号是固定的）先找到第n个孩子，然后看是否和E匹配
    2. nth-of-type：对父类元素里面指定子元素进行排序选择；先去匹配E，然后再根据E找第n个孩子

伪元素选择器（重点）

  1. 伪元素选择器可以帮助我们利用CSS创建新标签元素，而不需要HTML标签，从而简化HTML结构
    选择符 简介
    ::before 在元素内部的前面插入元素
    ::after 在元素内部的后面插入元素
  2. 注意：
    1. before和after创建一个元素，但是属于行内元素
    2. 新创建的这个元素再文档树中是找不到的，所以我们称为伪元素
    3. 语法：element::before{}
    4. before和after必须有content属性
    5. before在父元素内容的前面创建元素，after在父元素内容的后面插入元素
    6. 伪元素选择器和标签选择器一样，权重为1
  3. 伪元素使用场景
    1. 伪元素选择器使用场景3：伪元素清除浮动
      • after伪元素清除浮动
      • 双伪元素清除浮动

CSS3盒子模型(重点)

  1. CSS3中可以通过box-sizing来指定盒模型，有两个值:
    • box-sizing: content-box 盒子大小为width + padding + border(默认，W3C标准盒子模型)
    • box-sizing: border-box 盒子大小为width(即padding和border不会撑大盒子)

CSS3其他特性

CSS3滤镜filter

  1. filter: CSS属性将模糊或颜色偏移等图形效果应用于元素
  2. 语法: filter: 函数();
    • (此函数为虚化函数)
      <style>
  div img {
      filter: blur(10px);
  }
</style>

CSS3 calc函数

  1. calc(): 此函数用于在声明CSS属性值时执行一些计算
  2. 例: 子盒子的宽高永远比父盒子小30px
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .father {
            width: 300px;
            height: 170px;
            background-color: skyblue;
        }
        .son {
            width: calc(100% - 30px);
            height: calc(100% - 30px);
            background-color: pink;
        }
    </style>
</head>
<body>
    <div class="father">
        <div class="son"></div>
    </div>
</body>
</html>

边框图片

  1. 边框图片使用场景：盒子大小不一，但是边框样式相同，此时就需要边框图片来完成
  2. 为了实现丰富多彩的边框效果，在CSS3中，新增了border-image属性，这个新属性允许指定一幅图像作为元素的边框
  3. 边框图片切图原理（重要）：九宫格切法
  4. 切图顺序：上右下左
  5. 边框图片使用语法（重要）
    1. 属性
      属性 描述
      border-image-source 用在边框的图片的路径
      border-image-slice 图片边框向内偏移（裁剪的尺寸，一定不加单位，上右下左顺序）
      border-image-width 图片边框的宽度（需要加单位），不是边框的宽度而是边框图片的宽度，边框图片的宽度不会挤压文字，默认为边框的宽度
      border-image-repeat 图像边框是否应平铺（repeat）、铺满（round）或拉伸（stretch）默认拉伸 
      <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        div {
            width: 300px;
            height: 300px;
            border: 2px solid pink;
            /* 边框图片的路径 */
            border-image-source: url(img/边框图片.png);
            /* 裁剪边框图片 */
            border-image-slice: 49 49 49 49;
            /* 图片边框的宽度 */
            border-image-width: 10;
            /* 图像边框是否应平铺（repeat）、铺满（round）或拉伸（stretch） */
            border-image-repeat: stretch;
        }
    </style>
</head>
<body>
    <div></div>
</body>
</html>

background: -webkit-linear-gradient(left, pink, skyblue);

CSS3过渡特性(重点)

  1. 过渡(transition)是CSS3中具有颠覆性的特征之一,我们可以在不使用Flash动画或javaScript的情况下,当元素从一种样式变换为另一种样式时为元素添加效果
  2. 过渡动画: 是从一个状态渐渐的过渡到另外一个状态
  3. 经常和:hover一起搭配使用
  4. 属性值
    1. 属性: 想要变化的CSS属性，宽度，高度，背景，颜色，内外边距都可以。如果所有的属性都需要变化过渡,则属性值为all
    2. 花费时间: 单位是秒(必须写单位) 比如0.5s
    3. 运动曲线: 默认是ease(可以省略)
    4. 何时开始: 单位是秒(必须写单位)可以设置延迟触发事件,默认为0s(可以省略)
  5. 语法: 过渡属性 花费时间 运动曲线 何时开始;
  6. 过渡口诀: 谁做过渡(动画)给谁加
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        div {
            width: 200px;
            height: 80px;
            background-color: pink;
            /* transition: width 1s; */
            /* 想要写多个属性，则用逗号进行分隔 */
            transition: width 1s, hight 1s;
            /* 所有的属性都需要变化过渡,则属性值为all,开发中常用 */
            transition: all 1s;
            border-top-right-radius: 100px;
            border-bottom-right-radius: 100px;
        }
        div:hover {
            width: 700px;
            height: 300px;
            background-color: skyblue;
        }
    </style>
</head>
<body>
    <div></div>
</body>
</html>

2D转换

  1. transform是CSS3中具有颠覆性的特征之一，可以实现多元素的位移，旋转缩放等效果
    1. 移动：translate
    2. 旋转：rotate
    3. 缩放：scale

二维坐标系

  1. 2D转换是改变在二维平面上的位置和1形状的一种技术
  2. 网页二维坐标系：x轴正轴水平向右，y轴正轴水平向下

转换之移动translate

  1. 2D移动还是2D转换里面的一种功能，可以改变元素在页面中的位置，类似定位
  2. 语法：
    1. transform：translate(x,y);
    2. 分开写：
      • transform: translateX(n);
      • transform: translateY(n);
  3. 例：
    <style>
    div {
        width: 200px;
        height: 200px;
        background-color: skyblue;
        /* transform：translate(x,y); */
        transform: translate(100px, 100px);
        /* transform: translateX(n); */
        transform: translateX(100px);
        /* transform: translateY(n); */
        transform: translateY(100px);
    }
</style>
  4. 重点：
    1. 定义2D转换中的移动，沿着X和Y轴移动元素
    2. translate最大的优点：不会影响到其他元素的位置
    3. translate中的百分比单位是相对于自身元素的translate:(50%, 50%);
    4. 对行内标签没有效果
  5. 小技巧：让盒子实现水平垂直居中
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        div {
            position: relative;
            width: 500px;
            height: 500px;
            background-color: skyblue;
        }
        p {
            position: absolute;
            /* 定位 */
            top: 50%;
            left: 50%;
            width: 100px;
            height: 100px;
            background-color: pink;
            /* 移动 */
            transform: translate(-50%, -50%);
        }
    </style>
</head>
<body>
    <div>
        <p></p>
    </div>
</body>
</html>

转换之2D旋转rotate

  1. 2D旋转是指让元素在2维平面内顺时针旋转或者逆时针旋转
  2. 语法
    • transform: rotate(度数)
  3. 重点
    1. rotate里面跟度数，单位是deg 比如: totate(45deg)
    2. 角度为正时，顺时针；角度为负时，为逆时针
    3. 默认旋转的中心点是元素的中心点
  4. 例：
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        img {
            width: 200px;
            height: 200px;
            border: 2px solid pink;
            border-radius: 100px;
            transition: all 1s;
        }
        img:hover {
            /* rotate旋转 */
            transform: rotate(360deg);
        }
    </style>
</head>
<body>
    <img src="../../images/fat.png" alt="">
</body>
</html>
  5. 2D转换中心点transform-origin
    1. 语法：
      • transform-origin: x y;
    2. 重点
      • 注意后面参数x和y要用空格隔开
      • x y 默认转换的中心点是元素的中心点(50% 50%)
      • 还可以给x y设置像素或者方位名词(top bottom left right center)
    3. 例
      <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        div {
            width: 100px;
            height: 100px;
            background-color: skyblue;
            margin: 200px auto;
            transition: all 1s;
            /* 旋转中心点为左下角 */
            transform-origin: left bottom;
            /* 旋转中心点为距离盒子上边框和左边框各30px */
            transform-origin: 30px 30px;
        }
        div:hover {
            transform: rotate(360deg)
        }
    </style>
</head>
<body>
    <div></div>
</body>
</html>

2D转换之缩放scale

  1. 缩放：可以将元素放大和缩小。只要给元素添加上这个属性就能控制它放大或缩小
  2. 语法
    • transform: scale(x, y);
  3. 注意
    1. 注意其中的x和y用逗号分隔
    2. transform: scale(1, 1)：宽和高都放大了一倍，相当于没有放大
    3. transform: scale(2, 2)：宽和高都放大了2倍
    4. transform: scale(2)：只写一个参数，第二个参数则和第一个参数一样，相当于scale(2, 2)
    5. transform: scale(0.5, 0.5)：缩小
    6. sacle缩放的最大优势：可以设置转换中心点缩放，默认以中心点缩放，而且不影响其他盒子
  4. 演示：
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        div {
            width: 200px;
            height: 200px;
            background-color: skyblue;
            margin: 100px auto;
            transition: all 1s;
        }
        div:hover {
            /* transform: scale(2, 2)：宽和高都放大了2倍 */
            /* transform: scale(2, 2); */
            /* transform: scale(2)：只写一个参数，第二个参数则和第一个参数一样，相当于scale(2, 2) */
            /* transform: scale(2); */
            /* transform: scale(0.5, 0.5)：缩小 */
            transform: scale(0.5, 0.5);
        }
    </style>
</head>
<body>
    <div></div>
</body>
</html>

2D转换综合写法

  1. 注意
    1. 同时使用多个转换，语法为：
      • ransform: translate() rotate() scale();
    2. 其顺序会影响转换的效果（先旋转会改变坐标轴的方向）
    3. 当我们同时有位移和其他属性的时候，记得要将位移放到最前面
      <style>
    div {
        width: 150px;
        height: 150px;
        background-color: pink;
        transition: all 0.5s;
    }
    div:hover {
        transform: translate(200px, 100px) rotate(180deg) scale(1.5);
    }
</style>

动画

  1. 动画是CSS3中具有颠覆性的特征之一，可以通过设置多个节点来精确控制一个或一组动画，常用来实现复杂的动画效果；相比于过度，动画可以实现更多变化，更多控制，连续自动播放等效果
  2. 制作动画分为两步
    1. 定义动画
    2. 使用（调用）动画

动画的基本使用

  1. 0%是动画的开始，100%是动画的完成。这样的规则就是动画序列
  2. 在@keyframes中规定某项CSS样式，就能创建由当前样式逐渐改为新样式的动画效果
  3. 动画是使元素从一种样式逐渐变化为另一种样式的效果，可以改变任意多的样式任意多的次数
  4. 请用百分比来规定变化发生的时间，或者关键词”from”和”to”，等同于0%和100%

用keyframes定义动画（类似定义类选择器）

```
@keyframes 动画名称 {
  0% {
    width: 100px;
  }
  100% {
    width: 200px;
  }
}
```

动画常见属性

属性 描述
@keyframes 规定动画
animation 所有动画属性的简写属性，除了animation-play-state属性
animation-name 规定@keyframes动画的名称（必须的）
animation-duration 规定动画完成一个周期所花费的秒或毫秒，默认是0（必须的）
animation-timing-function 规定动画的速度曲线，默认是“ease”，匀速“linear”
animation-delay 规定动画何时开始，默认是0
animation-iteration-count 规定动画被播放次数，默认是1，还有infinite(无限)—–iteration（匀速的）
animation-direction 规定动画是否在下一周期逆向播放，默认是“normal”，“alternate”为逆向播放
animation-play-state 规定动画是否正在运行或暂停。默认是运行“running”，暂停“paused”
animation-fill-mode 规定动画结束后状态，保持forward，回到起始backwards
  1. 例：
     <style>
   @keyframes move {
       0% {
           transform: translateX(0, 0);
       }
       100% {
           transform: translate(1000px);
       }
   }
   div {
       width: 100px;
       height: 100px;
       background-color: pink;
       /* @keyframes动画的名称 */
       animation-name: move;
       /* 规定动画完成一个周期所花费的秒或毫秒，默认是0 */
       animation-duration: 3s;
       /* 规定动画的速度曲线，默认是“ease” */
       animation-timing-function: ease;
       /* 规定动画何时开始，默认是0 */
       animation-delay: 2s;
       /* 规定动画被播放次数，默认是1，还有infinite(无限) */
       animation-iteration-count: 3;
       /* 规定动画是否在下一周期逆向播放，默认是“normal”，“alternate”为逆向播放 */
       animation-direction: alternate;
       /* 规定动画结束购状态，保持forward，回到起始backwards */
       animation-fill-mode: backwards;
   }
   div:hover {
       /* 规定动画是否正在运行或暂停。默认是运行“running”，暂停“paused” */
       animation-play-state: paused;
   }
 </style>

动画简写

  1. animation: 动画名称 持续时间 运动曲线 何时开始 播放次数 是否反方向 动画起始或者结束的状态;
  2. 注意
    1. 简写属性里面不包括animation-play-state（停止动画）
    2. 暂停动画：animation-play-state: puased; 经常和鼠标经过等其他配合使用
    3. 想要动画走回来，而不是直接跳回来: animation-direction: alternate;
    4. 盒子动画结束后，停在结束位置: animation-fill-mode: forwards;

速度曲线细节

  1. animation-timing-function: 规定动画的速度曲线，默认是“ease”
    值 描述
    liner 动画从头到尾的速度是相同的，匀速
    ease 默认。动画以低速开始，然后加快，在结束前变慢
    ease-in 动画以低速开始
    ease-in-out 动画以低速开始和结束
    steps() 指定了时间函数中的间隔数量（步长）

3D转换

三维坐标系

  1. 三维坐标是指的是立体空间，立体空间是由3个轴共同组成的
    1. x轴：水平向右—注意：x右边为正值，左边为负值
    2. y轴：垂直向下—注意：y往下为正值，往上为负值
    3. z轴：垂直屏幕—注意：z往外为正值，往内为负值

3D移动 translate3d

  1. 3D移动在2D移动的基础上多加了一个可以移动的方向，就是z轴方向
    1. transform: translateX(100px)：在x轴上移动
    2. transform: translateY(100px)：在y轴上移动
    3. transform: translateZ(100px)：在z轴上移动(注意：translateZ一般用px单位)
    4. transform: translate3d(x, y, z)：其中x, y, z分别指移动的轴的方向的距离
      <style>
  div {
      width: 100px;
      height: 100px;
      background-color: pink;
      margin: 200px auto;
      /* 在x轴上移动 */
      transform: translateX(100px);
      /* 在y轴上移动 */
      transform: translateY(100px);
      /* 在z轴上移动(注意：translateZ一般用px单位) */
      transform: translateZ(100px);
      /* 其中x, y, z分别指移动的轴的方向的距离 */
      transform: translate3d(50px, 50px, 50px)
  }
</style>

透视 perspective

  1. 近大远小
  2. 透视也称为视距：视距就是人的眼睛到屏幕的距离
  3. 透视的单位是像素
  4. 透视写在被观察元素的父盒子上面
    <style>
    body {
        /* 透视写在被观察元素的父盒子上面 */
        perspective: 200px;
    }
    div {
        width: 100px;
        height: 100px;
        background-color: pink;
        margin: 200px auto;
        transform: translateZ(100px);
    }
</style>
    1. d：视距（透视），视距就是一个距离，人的眼睛到屏幕的距离
    2. z：z轴，物体距离屏幕的距离，z轴越大（正值）我们看到的物体就愈大

3D转换 rotate3d

  1. 3D旋转指可以让元素在三维平面内沿着x轴，y轴，z轴或者自定义轴进行旋转
  2. 语法
  3. transform: rotateX(45deg)：沿着x轴正方向旋转45度
  4. transform: rotateY(45deg)：沿着y轴正方向旋转45度
  5. transform: rotateZ(45deg)：沿着z轴正方向旋转45度
  6. transform3d: rotate3d(x, y, z, deg)：沿着自定义轴旋转deg角度（了解即可）
- x y z表示旋转轴的矢量
  - 例：transform3d: rotate3d(1, 1, 0, 45deg)
  - 就是沿着x轴和y轴矢量方向的对角线旋转45deg
    ```
    <style>
      body {
          /* 透视写在被观察元素的父盒子上面 */
          perspective: 300px;
      }
      div {
          width: 320px;
          height: 250px;
          margin: 200px auto;
          transition: all 1s;
      }
      img {
          width: 100%;
          height: 100%;
      }
      div:hover {
          /* 沿着x轴正方向旋转45度 */
          transform: rotateX(360deg);
          /* 沿着y轴正方向旋转45度 */
          transform: rotateY(360deg);
          /* 沿着z轴正方向旋转45度 */
          transform: rotateZ(360deg);
      }
    </style>
    ```

3D呈现transform-style

  1. 控制子元素是否开启三维立体环境
  2. transfrom-style: flat;子元素不开启3d立体空间 默认的
  3. transform-style: preserve-3d; 子元素开启立体空间
  4. 代码写给父级，但是影响的是子盒子
  5. 这个属性很重要，后面必用
