对象的扩展

属性的简写

  1. 当属性名和属性值名称相同时，属性值可以不写，只写一个属性名即可

Obeject.keys()方法

  1. 获取对象的所有属性名或方法名（不包括原型的内容），返回一个数组
  2. 例：
    var obj = {
  name: "john",
  age: "21", 
  getName: function () { 
    alert(this.name)
  }
};
console.log(Object.keys(obj));                 // ["name", "age", "getName"] 
console.log(Object.keys(obj).length);          // 3 
console.log(Object.keys(["aa", "bb", "cc"]));  // ["0", "1", "2"] 
console.log(Object.keys("abcdef"));            // ["0", "1", "2", "3", "4", "5"]

Object.assign()方法

  1. assign 方法将多个原对象的属性和方法都合并到了目标对象上面
  2. 第一个参数是目标对象，后面的都是源对象
  3. 例：
    var target = {}; //目标对象 
var source1 = {name : 'ming', age: '19'}; //源对象 1 
var source2 = {sex : '女'}; //源对象2 
var source3 = {sex : '男'}; //源对象3，和source2 中的对象有同名属性sex ，则覆盖同名属性sex
Object.assign(target, source1, source2, source3); 
console.log(target); //{name : 'ming', age: '19', sex: '男'}
