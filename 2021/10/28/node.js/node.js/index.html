Node.js

Node.js概述

  1. node.js不是一门语言、不是库、不是框架，是一个JavaScript运行时环境

    1. node.js可以解析和执行JavaScript代码，让JavaScript可以完全脱离浏览器来运行
    2. 在node.js这个JavaScript执行环境中为JavaScript提供了一些服务器级别的操作API，例如：
      • 文件读写
      • 网络服务的构建
      • 网络通信
      • http服务器
      • …

  2. 作用：让JavaScript成为与PHP、Python等平起平坐的语言

  3. 目的：学习node.js来打开服务器

  4. 概括：前段脱离后端，直接通过JS写项目

  5. 官网下载： nodejs.org

    1. 中文网址：nodejs.cn
    2. 学习社区：condejs.org

Node.js能做什么

  1. Web服务器后台
  2. 命令行工具
    • npm（node.js开发）
    • git（c语言开发）
    • hexo（node.js开发）
    • …
  3. 对于前端开发工程师而言，接触node.js最多的是它的命令行工具，自己写的很少只要是使用第三方的
    • webpack
    • gulp
    • npm

Node.js基础操作（REPL）

简介

  1. R：read – 读取（读取用户输入的JS代码）
  2. E：exec – 执行（执行用户输入的JS代码）
  3. P：print – 打印（打印用户输入的JS代码）
  4. L：loop – 循环（后续所有JS代码执行都会循环以上命令）
  5. 退出（按两下ctrl + c）
  6. 清屏：cls

内置模块

os操作系统信息模块

  1. 引入node.js内置os模块

    var os = require('os');

  2. 方法调用可以查询node.js帮助文档

path路径处理模块

  1. 引入node.js内置path模块

    var http = require('http');

  2. **dirname(path)**：去掉文件路径的最后一层

  3. **basename(path)**：取最后一层

    1. path：文件路径

url网址处理模块

  1. 引入node.js内置url模块

    var url = require('url');

  2. **parse(urlString, boolean)**：接受网址字符串，解析并返回网址对象

    1. urlString：网址字符串

    2. boolean：值为true或false，当值为true时，输出url网址对象的query值为对象形式，更方便提取数据

fs文件系统模块

  1. 引入node.js内置fs模块

    var fs = require('fs');

  2. **fs.readFile(path[, options], callback)**：异步的读取文件的全部内容

    1. path：文件或文件名描述符

    2. options：指定编码格式，如utf8

    3. callback：回调函数，有两个参数

      • err：错误内容

        • 写入成功，err的值为null
        • 写入失败，err的值为错误对象

      • data：数据

  3. **fs.writeFile(file, data[, options], callback)**：异步的写入文件内容

    1. file：文件名或文件路径

    2. data：写入的文件内容

    3. options：指定的编码格式，如utf8

    4. callback：回调函数，有一个参数

      • err：错误内容

        • 写入成功，err的值为null
        • 写入失败，err的值为错误对象

http服务器模块

创建http服务器

  1. 引入node.js内置http模块

    var http = require('http');

  2. 创建web服务器，返回一个server实例

    var server = http.createServer();

  3. 监听请求

    1. 注册request请求事件，当客户端请求过来，就会自动触发服务器的request请求事件，然后执行第二个参数：回调函数

    2. 回调函数参数

      • request：请求
      • response：响应
      server.on('request', function(request, response){
    console.log('收到客户端请求');
})

  4. 启动服务

    1. 绑定端口号，启动服务器

      server.listen(3000, function() {
    console.log('服务器启动成功，可以通过 http://127.0.0.1:3000/ 来进行访问');
})

      • 将数组转换为字符串 – JSON.stringify(arr);

请求和响应对象

  1. 请求对象

    // 监听请求
// resquest：请求    response：响应
server.on('request', (request, response) => {
  console.log(request.headers);     // 获取请求头信息（对象）
  console.log(request.rawHeaders);  // 获取请求头信息（数组）
  console.log(request.httpVersion); // 获取http版本
  console.log(request.method);      // 获取请求方法
  console.log(request.url);         // 获取请求路径（注：不含网址）

  // 解决乱码问题
  response.setHeader('Content-type', 'text/html;charset=utf-8')
  // 响应数据结果
  response.write('打印request对象');
  // 响应结束
  response.end();
})

    1. 主要使用以下两个方法

      console.log(request.method);      // 获取请求方法
  console.log(request.url);         // 获取请求路径（注：不含网址）

  2. 响应对象

    // 监听请求
// resquest：请求    response：响应
server.on('request', (request, response) => {
  // 有请求就有响应
  // 修改响应状态码
  response.statusCode = 404;
  // 修改响应状态信息
  response.statusMessage = 'Not Found';

  // 解决乱码问题
  response.setHeader('Content-type', 'text/html;charset=utf-8')
  // 响应数据结果
  response.write('打印response对象');
  // 响应结束
  response.end();
})

    1. 主要使用以下两个方法

        response.statusCode = 404;  // 修改响应状态码
  response.statusMessage = 'Not Found';
