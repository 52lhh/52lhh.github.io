闭包

变量作用域

  1. 变量根据作用域的不同分为两种：全局变量和局部变量
  2. 函数内部可以使用全局变量
  3. 函数外部不可以使用局部变量
  4. 当函数执行完毕后，本作用域内的局部变量会被销毁

什么是闭包

  1. 闭包（closure）指有权访问另一个函数作用域中变量的函数
  2. 简单理解：一个作用域可以访问另外一个函数内部的局部变量，则这个被访问的局部变量所在的函数就是闭包
  3. 闭包的主要作用：延伸了变量的作用范围
  4. 闭包是一种保护私有变量的机制，在函数执行时形成私有的作用域，保护里面的私有变量不受外界干扰
  5. 直观的说就是形成一个不销毁的栈环境

闭包应用

  1. 循环注册点击事件

    1. 经典面试题：利用闭包的方式得到当前小li的索引号
      <body>
<li>1</li>
<li>2</li>
<li>3</li>
<li>4</li>
<script>
  var lis = document.querySelectorAll('li');
  for (var i = 0; i < lis.length; i++) {
      // 利用for循环创建了四个立即执行函数
      // 立即执行函数是一个小闭包，因为立即执行函数里面任何一个函数都可以使用它的i变量 
      (function (i) {
          lis[i].onclick = function () {
              console.log(i);
          }
      })(i);
  }
</script>
</body>

  2. 循环中的setTimeout()

    <body>
 <li>1</li>
 <li>2</li>
 <li>3</li>
 <li>4</li>
 <script>
     var lis = document.querySelectorAll('li');
     // for循环为同步任务
     for (var i = 0; i < lis.length; i++) {
         (function (i) {
             // 定时器为异步任务
             setTimeout(function () {
                 console.log(i);
             }, 2000)
         })(i);
     }
 </script>
</body>
