浅拷贝和深拷贝

学习笔记：浅拷贝和深拷贝

浅拷贝和深拷贝

浅拷贝

  1. 浅拷贝只是拷贝一层，更深层次对象级别的只拷贝引用
  2. 浅拷贝原理
    1. 把原来对象里面的复杂数据类型的地址拷贝给目标对象
    2. 会覆盖目标对象的里面的数据
    3. 目标拷贝对象里面的值会随着原来对象的改变而改变，因为他们指向同一个地址
  3. Object.assign(targer, sources) — es6新增方法可以浅拷贝
    1. targer：拷贝的目标对象
    2. sources：拷贝谁

深拷贝

  1. 深拷贝拷贝多层，每一级别的数据都会拷贝

  2. 深拷贝原理

    1. 把原来对象里面的数据完全复制一份给目标对象
    2. 会覆盖目标对象的里面的数据
    3. 若其中有不冲突的属性，会合并到一起，比如目标对象中的对象的属性age和原来对象中的对象的属性name
    4. 目标拷贝对象里面的值不会随着原来对象的改变而改变

  3. 代码实现深拷贝

    script>
    var obj = {
        name: '熊大',
        sex: '男',
        age: {
            one: 7,
            two: 6
        },
        friend: ['吉吉', '毛毛', '蹦蹦']
    }
    var o = {}

    // 封装深拷贝函数
    function deepClone(newObj, oldObj) {
        for (var k in oldObj) {
            // 获取属性值，k为属性名，oldObj[k]为属性值
            var item = oldObj[k];
            // 当属性值为数组时
            // 判断数组必须放在判断对象之前，因为数组也为对象
            if (item instanceof Array) {
                // newObj[k]为新对象的属性值
                newObj[k] = [];
                deepClone(newObj[k], item);
            } else if (item instanceof Object) {
                // 当属性值为对象时
                newObj[k] = {};
                deepClone(newObj[k], item);
            } else {
                // 当属性值为简单数据类型时
                newObj[k] = item;
            }
        }
    }
    deepClone(o, obj);
    console.log(o);
</script>
