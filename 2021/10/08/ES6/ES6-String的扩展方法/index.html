String的扩展方法

模板字符串

  1. ES6新增的创建字符串的方式，使用反引号定义

  2. 模板字符串语法：${}

  3. 模板字符串的特点

    1. 模板字符串中可以解析变量

      let name = `张三`;
let sayHello = `你好啊，我的朋友${name}`;
console.log(sayHello); // 你好啊，我的朋友张三

    2. 模板字符串中可以实现换行

    3. 在模板字符串中可以调用函数

      let fn = () => {
    return `我爱你~`;
}
let love = `胖崽崽，${fn()}`;
console.log(love);

实例方法：startsWith()

  1. startsWith()：表示参数字符串是否在原字符串的头部，返回布尔值

实例方法：endsWith()

  1. endsWith()：表示参数字符串是否在原字符串的尾部，返回布尔值
    <script>
  let str = 'hello world!';
  var flag1 = str.startsWith('hell'); // true
  var flag2 = str.endsWith('d!'); // true
  console.log(flag1, flag2);
</script>

实例方法：repeat()

  1. repeat()：表示将原字符串重复n次，返回一个新的字符串
    console.log('我爱憨崽'.repeat(5));
