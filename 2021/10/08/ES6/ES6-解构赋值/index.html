ES6-解构赋值

发表于 2021-10-08 更新于 2021-10-14 分类于 ES6 阅读次数：

学习笔记：ES6-解构赋值
个人笔记可能会存在一些错误，您在阅读中如若发现，欢迎您的指正

解构赋值

  1. ES6中允许从数组中提取值，按照对应位置，对变量赋值，对象也可以实现解构

数组解构

  1. 数组解构允许按照一一对应的关系从数组中提取值，然后将值赋值给变量

  2. 如果解构不成功，变量的值为undefined

    <script>
  let arr = [1, 'a', 2];
  // 按照一一对应的关系从数组中提取值，然后将值赋值给变量
  let [a, b, c] = arr;
  console.log(a); // 1
  console.log(b); // a
  console.log(c); // 2
</script>

    <script>
  let arr = [1, 'a', 2];
  // 按照一一对应的关系从数组中提取值，然后将值赋值给变量
  let [a, b] = arr;
  console.log(a); // 1
  console.log(b); // a
</script>

    <script>
  let arr = [1, 'a', 2];
  // 按照一一对应的关系从数组中提取值，然后将值赋值给变量
  let [a, b, c, d] = arr;
  console.log(a); // 1
  console.log(b); // a
  console.log(c); // 2
  console.log(d); //undefined
</script>

对象解构

  1. 对象解构必须根据属性名进行匹配

  2. 如果解构不成功，变量的值为undefined

    <script>
  let person = {
    name: 'zhangsan',
    age: '不详'
  }
  // 对象解构
  let { name, age } = person;
  console.log(name); // zhangsan
  console.log(age); // 不详
</script>

    <script>
  let person = {
    name: 'zhangsan',
    age: '不详'
  }
  // 冒号左侧的变量只用于匹配，赋值给冒号右边的变量
  let { name: myname, age: myage } = person;
  // 当对象解构成功，属性匹配成功，则将属性对应的值赋值给冒号左侧的变量
  console.log(myname); // zhangsan
  console.log(myage); // 不详
</script>

    <script>
  let person = {
    name: 'zhangsan',
    age: '不详'
  }
  // 必须利用属性名进行匹配
  let { n, a } = person;
  console.log(n); // undefined
  console.log(a); // undefined
</script>
